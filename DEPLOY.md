# Deploy to Render.com (Free)

## Option 1: Via Render Dashboard (2 minutes)

1. Go to https://dashboard.render.com
2. Click "New +" → "Web Service"
3. Connect your GitHub account
4. Create a new repo with this code
5. Select it and deploy

Settings:
- Build Command: `npm install`
- Start Command: `npm start`
- Environment: Node

## Option 2: Via GitHub + Render Button

1. Create GitHub repo
2. Push this code
3. Click: [![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)
